# EasyNote v2 长期路线图 (Roadmap)

本文档记录了 EasyNote v2 的核心功能规划，旨在保持“极致好用、无感智能”的开发方向。

## 📍 已完成项目
- [x] 基于 React 19 + FastAPI 的全栈架构
- [x] 新拟态 (Neumorphism) UI 设计语言
- [x] 智能语义解析 (AI 快速创建任务)
- [x] PWA 支持 (支持安装至手机/桌面)
- [x] 移动端触觉反馈系统

## 🚀 正在进行
### 第一阶段：AI 智能复盘 (正在实施)
- **紫色灯语**: 每日上线自动生成一句“点睛之笔”洞察。
- **语义搜索**: 支持模糊语义的任务查找。

## 🔭 未来规划
### 深度复盘系统
- **人工交互**: 支持用户勾选特定任务/项目进行细化复盘。
- **多轮对话**: 引入 LangChain Memory，实现更深层、有记忆的 AI 对话。
- **复盘历史**: 建立历史库，支持按周、按月查看个人成长轨迹。

### 极致体验优化
- **个性化皮肤**: 基于新拟态的深色模式与多主题色。
- **桌面/锁屏组件**: 针对扫马、笔记等场景的 Widget 支持。
